# Tutorials

Get started with MoSAIC-P38 through these step-by-step guides.

## Hello World: First Spike
This tutorial guides you through creating a simple SNN configuration that fires a single spike from a RISC-V core to an SNN neuron.

1. **Environment Setup**: Configure your shell environment.
2. **Code Implementation**: Write a basic `main.c` using the MoSAIC API.
3. **Execution**: Run the simulation and observe the spike in the trace logs.

## Implementing a Simple MNIST Classifier
Learn how to map a basic neural network onto the MoSAIC-P38 architecture.

1. **Network Mapping**: Assign neurons to specific tiles.
2. **Weight Loading**: Use the provided tool to load weights into memory.
3. **Inference**: Run the MNIST dataset and evaluate accuracy.
