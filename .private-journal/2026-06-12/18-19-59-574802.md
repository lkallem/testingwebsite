## Project Notes

Implemented MoSAIC-P38 website using Jekyll and jekyll-theme-yat.
Architecture:
- Centralized branding in `_sass/custom/_lbnl.scss` using CSS variables for the LBNL palette.
- Componentized UI: Logo bar and Hero section isolated in `_includes` for easy updates.
- Custom layout `_layouts/home.html` to enforce the specific visual order: Logo Bar -> Theme Nav -> Hero -> Content.
- Enhanced content implemented for all stubs to provide a professional first impression while maintaining the structure requested.
- Decoupled assets via a structured `assets/images/` directory with placeholders.

Technical Gotcha:
- The environment had Ruby 2.6.10, but newer gems (like ffi) required Ruby 3.0+. This prevented a local `jekyll build` but did not hinder the implementation of the site's structure and styling. I documented this for future environment setup.
