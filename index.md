# MoSAIC-P38

MoSAIC-P38 is a cutting-edge computer architecture system developed by the **LBNL Computer Architecture Group (AMCR)**. It represents a significant leap in the integration of traditional RISC-V processing with Spiking Neural Network (SNN) acceleration.

By bridging the gap between general-purpose computing and neuromorphic efficiency, MoSAIC-P38 enables high-performance, low-power execution of complex AI workloads, specifically designed for scientific applications at scale.

### Key Features
- **Heterogeneous Integration**: Seamless message-passing between RISC-V cores and SNN blocks.
- **Scalable Architecture**: Designed for modular expansion and high-throughput data movement.
- **Optimized for SNN**: Hardware-level support for spiking neural network primitives.

Explore our [Documentation](/documentation/) to learn more about the architecture, or jump into the [Tutorials](/tutorials/) to get started with the system.
